Button Hover Demo

Sommer 2022

Hintergrund

Eine Interaktive Demo verschiedener Hover-Effekte für Buttons.

An einem von Langeweile geplagten Nachmittag habe ich diese kleine Seite geschrieben, in der man eine Einführung in die Möglichkeiten für Übergänge mit CSS bekommen sollte.

Einsteigerfreundlich kann man aus einer Palette von Klassen einen gewünschten Hover-Effekt zusammenstellen, die Farbe festlegen und den Code in sein Projekt übernehmen.

Natürlich habe ich versucht, alles so gut wie möglich selbst zu machen - das hieß allerdings auch das Syntax-Highlighting im Editor-Fenster, das letztendlich mit RegEx gelöst wurde. Nicht die beste Idee, aber reguläre Ausdrücke hatte ich zu dem Zeitpunkt schon länger nicht mehr verwendet.

Die Effektivität dieses Tutorials wurde noch nicht wirklich getestet, aber als ausbaufähiges Nebenprojekt könnte ich mir vorstellen, über die Zeit weitere Kapitel hinzuzufügen.
