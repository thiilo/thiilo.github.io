SignageHub

Sommer 2022

Hintergrund

Eine customisierte Signage-Lösung für das Restaurant eines Freundes.

SignageHub ermöglicht die Steuerung von einer Anzahl externer Monitore, die als Kiosks Inhalte darstellen. Es können von jedem Geruat aus live Inhalte geändert oder Wechsel zeitplanmäßig automatisch durchgeführt werden.

Bei diesem Projekt habe ich mit Marc Hohl zusammen das Front-End und die UX mit PHP und CSS (hauptsächlich Bootstrap) gestaltet.

Gleichzeitig wurden zwei Rechner für das Restaurant konfiguriert, ein Büro- und Steuerungsrechner und ein Server, auf dem SignageHub mit Apache gehostet und die angeschlossenen Displays per i3 verwaltet werden.
